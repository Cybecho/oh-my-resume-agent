# 채용 의도 분석 출력 스키마

`03_의도분석.md` 파일은 아래 형식을 따른다.

---

```markdown
# 채용 의도 분석: {회사명} {직무}

## 채용 배경 추론

- 추정 배경: (결원충원 / 신규사업 확장 / 조직 강화 / 기타)
- 근거: ...
- 긴급도 추정: (높음/중간/낮음)

## 핵심 채용 의도

이 포지션을 뽑는 진짜 이유:
- ...

## 서류 평가 기준 추론

평가자가 자소서를 읽을 때 가장 먼저 보는 것:
1. ...
2. ...

## 자소서 문항별 분석

### 문항 1: "{문항 전문}"

- **문항 유형**: (지원동기 / 성장과정 / 장단점 / 입사후포부 / 직무역량 / 도전·실패)
- **출제 의도**: 이 질문을 통해 확인하려는 것
- **평가 역량**: [역량1, 역량2]
- **기대 답변 프레임**: 해당 유형의 기대 공식에 맞춘 좋은 답변의 구조/방향
- **함정/주의사항**: 이렇게 쓰면 감점 (소극적 동기 함정 등 유형별 주의사항 포함)
- **차별화 포인트**: 다른 지원자와 구분되려면

### 문항 2: "{문항 전문}"
(동일 구조 반복)

...

## 문항 간 연결 전략

- 문항들이 전체적으로 평가하려는 큰 그림: ...
- 문항 간 중복 위험 영역: ...
- 각 문항에서 차별적으로 보여줘야 할 것: ...
```
