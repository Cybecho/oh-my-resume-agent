# EN Resume Plan Schema

`en_resume/04_resume_plan.md` uses the following structure.

```markdown
# Resume Plan: {Company} - {Role}

## Global Constraints
- target_region: US|CA
- page_guideline: 1-page preferred (early-career)
- variant_count: 3
- include_cover_letter: false

## Section Strategy
- Header
- Professional Summary
- Skills
- Experience
- Projects
- Education
- Certifications / Community (optional)

## Variant Lens Design

### Variant A (JD-dense)
- Positioning statement:
- Keyword priorities:
- Experience priorities:
- Risks:

### Variant B (impact-dense)
- Positioning statement:
- Keyword priorities:
- Experience priorities:
- Risks:

### Variant C (balanced)
- Positioning statement:
- Keyword priorities:
- Experience priorities:
- Risks:

## Bullet Rules
- Structure: Action Verb + Context + Result
- Tense: present for current role, past for previous roles
- No weak opener: "responsible for", "duties included"
- No unsupported numbers

## Density Guidance
- Keep concise scannable bullets.
- Prioritize high-evidence claims and role-relevant impact.
```
