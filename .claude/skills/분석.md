# 분석 — Step 2-3 단독 재실행

JD 분석 + 채용 의도 추론 단계를 단독으로 재실행한다.

## 사전 조건

1. `output/` 폴더에서 대상 회사 폴더를 확인한다
   - 여러 개면 사용자에게 선택 요청
2. 해당 폴더의 `state.json`을 읽는다
3. `01_기업조사.md`가 존재하는지 확인 (없으면 `/조사`를 먼저 실행하라고 안내)

## 실행

1. `agents/JD분석선생.md`를 Read로 읽는다
2. state.json에서 JD 텍스트, 01_기업조사.md 읽기
3. JD 분석 수행 → `02_JD분석.md` 덮어쓰기 저장

4. `agents/인사관선생.md`를 Read로 읽는다
5. 01_기업조사.md + 02_JD분석.md + 문항 참조
6. 채용 의도 + 문항 의도 분석 → `03_의도분석.md` 덮어쓰기 저장

7. `state.json` 업데이트: step 2,3 → completed

## 완료 후

- 주요 분석 결과 요약 (필수역량 TOP 3, 핵심 채용 의도)
- "이후 단계를 재실행하려면 `/매칭`을 사용하세요."
