# 조사 — Step 1 단독 재실행

기업 조사 단계만 단독으로 재실행한다.

## 사전 조건

1. `output/` 폴더에서 대상 회사 폴더를 확인한다
   - 여러 개면 사용자에게 선택 요청
   - 없으면: 회사명을 물어보고, `/자소서`를 먼저 실행하라고 안내
2. 해당 폴더의 `state.json`을 읽어 현재 상태 확인

## 실행

1. `agents/조사선생.md`를 Read로 읽는다
2. 조사선생 지침에 따라 기업 리서치 수행 (WebSearch, WebFetch)
3. Culture-Sync 어휘 추출 포함
4. `output/{YYYYMMDD}_{회사명}/01_기업조사.md`를 덮어쓰기 저장
5. `state.json` 업데이트: step 1 → completed, updated_at 갱신

## 완료 후

- 변경된 조사 결과 요약 표시
- "이후 단계(분석~검수)도 재실행하시겠습니까? `/분석`으로 이어서 진행할 수 있습니다."
