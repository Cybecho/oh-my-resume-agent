# 작성 — Step 7 단독 재실행

자소서 작성 단계를 단독으로 재실행한다.

## 사전 조건

1. `output/` 폴더에서 대상 회사 폴더를 확인한다
   - 여러 개면 사용자에게 선택 요청
2. 해당 폴더의 `state.json`을 읽는다
3. `06_기획.md`와 `05_경험매칭.md`가 존재하는지 확인

## 실행

1. `agents/필체선생.md`를 Read로 읽는다
2. 06_기획.md, 05_경험매칭.md 읽기
3. `data/writing_samples/` 전체 읽기 (문체 학습)
4. 경험 원문 읽기 (`data/experience_cards/` 및 `data/experiences/`, Notion)
5. state.json에서 문항 + 글자수 확인
6. 3세트 자소서 작성
7. 저장:
   - `07_자소서/컨셉1.md` 덮어쓰기
   - `07_자소서/컨셉2.md` 덮어쓰기
   - `07_자소서/컨셉3.md` 덮어쓰기
8. `state.json` 업데이트: step 7 → completed

## 특수: 특정 컨셉만 재작성

사용자가 "컨셉2만 다시 써줘"와 같이 요청한 경우:
- 해당 컨셉의 기획 내용만 참조하여 재작성
- 해당 파일만 덮어쓰기

## 완료 후

- 각 컨셉 첫 문장 미리보기 표시
- "검수를 실행하려면 `/검수`를 사용하세요."
